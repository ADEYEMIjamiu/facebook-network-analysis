# Facebook Network Analysis

This project analyses a real-world Facebook social network dataset using Python and NetworkX.

## Contents
- Network construction and visualisation
- Degree distribution analysis
- Clustering and path analysis
- Centrality measures
- Comparison with ER, BA, and WS models

## Dataset
The dataset is sourced from the Stanford Network Analysis Project (SNAP):
https://snap.stanford.edu/data/ego-Facebook.html

## Tools Used
- Python
- NetworkX
- Pandas
- Matplotlib

## Files
- `network_analysis.ipynb` – full code
- `report.pdf` – final report
